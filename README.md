# Установка и запуск проекта

## Установить зависимости

```bash
npm install
```

## Запустить проект

```bash
npm run dev
```

Проект откроется в браузере по адресу: [http://localhost:5173](http://localhost:5173)

## Сборка проекта

```bash
npm run build
```

Готовая версия появится в папке `dist/`.
